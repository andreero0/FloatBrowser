# Product Overview

Min is a fast, minimal web browser built on Electron that prioritizes privacy and distraction-free browsing.

## Core Features

- Full-text search for browsing history
- Built-in ad and tracker blocking
- Automatic reader view for articles
- Tasks system (tab groups for organizing work)
- Bookmark management with tagging
- Password manager integration (Bitwarden, 1Password, Keychain)
- Dark mode support
- Page translation
- Focus mode (prevents tab/task switching)

## Target Users

Users who value privacy, minimal UI design, and distraction-free browsing. The browser emphasizes simplicity over feature bloat.

## Platform Support

- macOS (Intel and ARM)
- Windows
- Linux (Debian, RedHat, AppImage)
- Raspberry Pi (32-bit and 64-bit ARM)
