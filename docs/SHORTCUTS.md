# Float Browser Keyboard Shortcuts Reference

Complete keyboard shortcuts for Float Browser, including both Float-specific shortcuts and all Min Browser shortcuts.

## Quick Reference Card

### Float Browser Shortcuts (Global)

These shortcuts work from any application:

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+F` | Toggle Float Browser visibility |
| `Cmd+Shift+A` | Toggle always-on-top mode |
| `Cmd+Shift+P` | Toggle Picture-in-Picture mode |
| `Cmd+1` | Apply Small window profile |
| `Cmd+2` | Apply Medium window profile |
| `Cmd+3` | Apply Large window profile |

---

## Float Browser Shortcuts (Detailed)

### Window Management

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd+Shift+F` | Toggle Visibility | Show or hide Float Browser window (works globally) |
| `Cmd+Shift+A` | Toggle Always-On-Top | Enable/disable always-on-top mode (works globally) |
| `Cmd+Shift+P` | Toggle PIP Mode | Enter/exit Picture-in-Picture mode (works globally) |

### Window Profiles

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd+1` | Small Profile | Apply Small profile (400×300, 80% opacity, always-on-top) |
| `Cmd+2` | Medium Profile | Apply Medium profile (800×600, 90% opacity, always-on-top) |
| `Cmd+3` | Large Profile | Apply Large profile (1200×800, 100% opacity, normal) |

### Opacity Control

| Action | Method |
|--------|--------|
| Adjust Opacity | Use the opacity slider in the toolbar |
| Quick Presets | Use Float menu → Opacity → Select percentage |

## Shortcut Conflicts

### Fullscreen vs Float Visibility

Both Min Browser and Float Browser use `Cmd+Shift+F`:
- **Min Browser**: Toggle fullscreen
- **Float Browser**: Toggle window visibility (global)

**Resolution**: Float Browser's global visibility toggle takes precedence. To enter fullscreen mode, use the View menu instead.

### Tab Switching vs Window Profiles

- `Cmd+[1-9]` switches between tabs in the current task
- `Cmd+1/2/3` applies window profiles

**Resolution**: Window profiles only use `Cmd+1`, `Cmd+2`, and `Cmd+3`. `Cmd+4` through `Cmd+9` remain available for tab switching.

## Customizing Shortcuts

### Float Browser Shortcuts

You can customize Float Browser global shortcuts:

1. Open Settings (`Cmd+,`)
2. Scroll to **Float Browser** section
3. Click **Configure** next to Global Shortcuts
4. Click on a shortcut to change it
5. Press your desired key combination
6. Click **Save**

### Min Browser Shortcuts

Min Browser shortcuts are currently not customizable. See the [Min Browser wiki](https://github.com/minbrowser/min/wiki) for more information.

## Platform-Specific Notes

### macOS

- `Cmd` = Command key (⌘)
- All shortcuts listed use macOS conventions
- Global shortcuts work across all Spaces and applications

### Keyboard Symbols

- `Cmd` = Command (⌘)
- `Shift` = Shift (⇧)
- `Option` = Option/Alt (⌥)
- `Ctrl` = Control (⌃)

## Tips for Efficient Keyboard Use

### Float Browser Workflow

**Quick Reference Workflow**
1. `Cmd+Shift+F` - Show Float Browser
2. `Cmd+2` - Apply Medium profile
3. `Cmd+L` - Focus address bar
4. Type search/URL
5. `Cmd+Shift+A` - Enable always-on-top
6. Click back to your work - Float Browser stays visible

**Video Watching Workflow**
1. `Cmd+Shift+P` - Enable PIP mode
2. Navigate to video
3. Adjust opacity with slider if needed
4. Float Browser stays on top automatically

**Multi-Tab Reference**
1. `Cmd+T` - Open new tabs for multiple references
2. `Cmd+[1-9]` - Switch between reference tabs
3. `Cmd+2` - Apply Medium profile
4. `Cmd+Shift+A` - Enable always-on-top

### Min Browser Workflow

**Research Workflow**
1. `Cmd+N` - Create new task for research topic
2. `Cmd+T` - Open multiple tabs
3. `Cmd+D` - Bookmark important pages
4. `Cmd+E` - Switch between tasks

**Reading Workflow**
1. `Cmd+Shift+R` - Enable reader view
2. `Cmd+Plus/Minus` - Adjust text size
3. `Cmd+F` - Search within page
