## Quick Start

1. **Install Float Browser**: See the [main README](../README.md) for installation instructions
2. **First Launch**: The welcome screen will guide you through Float Browser's features
3. **Learn Shortcuts**: Press `Cmd+Shift+F` to toggle visibility, `Cmd+Shift+A` for always-on-top
4. **Explore Features**: Try the opacity slider, PIP mode, and window profiles

## Essential Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+F` | Toggle Float Browser visibility |
| `Cmd+Shift+A` | Toggle always-on-top mode |
| `Cmd+Shift+P` | Toggle Picture-in-Picture mode |
| `Cmd+1/2/3` | Apply window profiles |
| `Cmd+L` | Focus address bar |
| `Cmd+T` | New tab |

## Common Use Cases

### Reference Browsing
1. Set opacity to 70-80%
2. Enable always-on-top (`Cmd+Shift+A`)
3. Apply Medium profile (`Cmd+2`)
4. Position alongside your work

### Video Watching
1. Enable PIP mode (`Cmd+Shift+P`)
2. Adjust opacity as needed
3. Position in a corner
4. Always-on-top is automatic

### Quick Access
1. Use `Cmd+Shift+F` to show/hide Float Browser
2. Keep it configured with your preferred settings
3. Access web content without disrupting workflow

## About Float Browser

Float Browser is built on [Min Browser](https://github.com/minbrowser/min) and includes all its features plus unique window management capabilities for macOS.

**Key Features**:
- Window transparency (30-100%)
- Always-on-top mode
- Picture-in-Picture mode
- Window profiles
- Global shortcuts
- All Min Browser features
