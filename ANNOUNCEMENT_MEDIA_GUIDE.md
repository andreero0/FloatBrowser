### Video Production Checklist

Before publishing the demo video, verify:

- [ ] Resolution is 1080p or higher
- [ ] Frame rate is 60fps for smooth animations
- [ ] Audio is clear and balanced
- [ ] No background noise or distractions
- [ ] All UI interactions are smooth and deliberate
- [ ] Keyboard shortcuts are shown with overlays
- [ ] On-screen text is readable and well-timed
- [ ] Transitions are smooth and professional
- [ ] Video demonstrates real-world use cases
- [ ] Call to action is clear
- [ ] Video length is 3-4 minutes
- [ ] File size is optimized for web delivery
- [ ] Video is tested on different devices

---

### Video Recording Tips

1. **Screen Recording:**
   - Use QuickTime Player or ScreenFlow for high-quality recording
   - Record at 2x resolution and scale down for better quality
   - Use 60fps for smooth UI animations

2. **Cursor:**
   - Use a cursor highlighting tool for visibility
   - Move cursor deliberately and smoothly
   - Pause briefly before clicking to let viewers anticipate

3. **Keyboard Shortcuts:**
   - Use KeyCastr or similar to show keyboard shortcuts
   - Ensure keyboard overlay is clearly visible
   - Pause briefly after pressing shortcuts

4. **Pacing:**
   - Move deliberately, not too fast
   - Pause between actions to let viewers process
   - Allow 2-3 seconds for each feature demonstration

5. **Content:**
   - Use real, professional websites (not Lorem Ipsum)
   - Use realistic code examples
   - Show actual workflows, not contrived demos

6. **Audio:**
   - Record voiceover in a quiet environment
   - Use a quality microphone
   - Speak clearly and at a moderate pace
   - Add subtle background music if desired
